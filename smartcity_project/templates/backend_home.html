Sistem Informasi Kota IET

Selamat datang di aplikasi repositori terpusat penanganan keluhan dan pelaporan infrastruktur wilayah administrasi Kabupaten Pesawaran.

Tentang Aplikasi

Aplikasi Kota IET dirancang khusus sebagai proyek implementasi matakuliah Pemrograman Internet Semester 4 di Program Studi Teknologi Rekayasa Internet, Politeknik Negeri Lampung.

Sistem ini berfungsi menjembatani komunikasi antara masyarakat umum Kabupaten Pesawaran dengan pihak otoritas dinas terkait dalam menangani kerusakan jalan, tumpukan keluhan sampah, banjir drainase, hingga perbaikan fasilitas penerangan umum.

Hubungi Kami

Jika mengalami kendala teknis operasional server database atau pengujian REST API, silakan hubungi tim administrasi pusat data teknologi:

  • Institusi: Lab Jaringan & Pemrograman - Politeknik Negeri Lampung
  • Pengembang Sistem: Ikrom Safei (NPM: 24782076)
  • Lokasi Wilayah Fokus: Pusat Pemerintahan Kabupaten Pesawaran, Lampung
